What it does & doesn't

✓ Auto-converted

  • recipe → craftRecipe (inputs/outputs, alternatives, keep-tools, quantities)
  • Recipe names → unique IDs + Recipes.json display names
  • Item Type = X → ItemType = base:x
  • Recipe.GetItemTypes.X tool lists → tags[X]

⚠ Flagged for review

  • OnGiveXP → set xpAward = Skill:n manually
  • timedAction defaulted to Making
  • Sound / AnimNode / Prop1·2 placement
  • Removed item props (e.g. TeachedRecipes)

◌ Lua: rewritten where safe

  • Removed-event handlers → commented out + annotated
  • require() of 132 removed base files → commented + annotated
  • Verified 1:1 global renames applied
  • Removed/restructured APIs & overrides → flagged, not guessed
